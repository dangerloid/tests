* COS'È CSS

CSS (Cascading Style Sheets) è il linguaggio che permette di apportare modifiche grafiche al documento HTML.

* STRUTTURA DI UN FOGLIO DI STILE

#+begin_src css

	h1 { /* selettore */
	font-size: /* attributo */ 50px; /* valore */
	color: #f15c62;
	text-transform: uppercase;
	text-align: center;
}

#+end_src
